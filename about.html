Sobre o Covid-19 Portal


Esta página foi concebida como projeto da cadeira de Tecnologias de Internet 1.
Tendo em conta o panorama atual da pandemia, aproveito o tema, juntamente com uma API, para criar este portal dedicado ao Covid-19.
Com esta API, é possível obter os dados atualizados diariamente, sobre todos os países que comunicam os seus valores à Organização Mundial de Saúde.
